# equity-research-notebook
Automated equity research framework integrating market and fundamental data (Yahoo Finance, WRDS/Compustat). Incorporating ticker screening, portfolio optimization, comparables analysis, fundamental metrics, and macro factor correlations for systematic equity selection and portfolio construction.

# Market Data Analysis
- Historical price data retrieval (Yahoo Finance)
- Return calculations and time-series analysis
- Volatility and performance metrics
# Fundamental Analysis (WRDS / Compustat)
- Financial statement data integration
- Key ratios (profitability, leverage, efficiency, distress)
- Cross-sectional company comparison
# Ticker Screening
- Screening based on financial and performance metrics
- Customizable filtering criteria
# Risk and Factor Analysis
- Beta estimation and expected returns
- Market-relative performance analysis
- Macro factor correlation analysis
# Portfolio Optimization
- Mean-variance optimization
- Efficient frontier construction
- Risk-return trade-off analysis
# Comparables and Valuation
- Multiples-based valuation (P/E, EV/EBITDA, etc.)
- Peer group analysis
- Identification of relative mispricing
